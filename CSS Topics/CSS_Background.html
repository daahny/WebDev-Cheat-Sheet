CSS Backgrounds

div {
    background-color: black;
    opacity: 0.3;
}

background-color: color;

Specifies the background color of an element

This can be used in conjunction with opacity: 0-1;

div {
    background-image: url('panda.png');
}

background-image: url('image_location');

Specifies an image to use as the background of an element

div {
    background-repeat: no-repeat;
}

background-repeat: repeat/no-repeat;

Specifies if an image is repeated or not

Possible values are repeat, no-repeat, repeat-x, repeat-y

div {
    background-position: right top;
}

background-position: horizontal vertical;

Specifies the starting position of a background image

horizontal values include left center right

vertical values include top center bottom

div {
    background-attachment: fixed;
}

background-attachment: attachment;

Specifies whether a background image scrolls with the rest of the page

values are scroll (default), fixed

div {
    background: color url('') 
                repeat contain 
                right top fixed;
}

background: everything;

Shorthand notation for all background properties
