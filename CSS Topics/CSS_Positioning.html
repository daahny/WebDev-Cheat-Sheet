CSS Positioning

Positioning

  • Static
    Static positioning is the default position for elements. Statically positioned elements are a part of the document flow and aren't moved with CSS properties. 
  • Relative
    Relative positioning is used to nudge an element out of its normal position. The element is still a part of document flow and isn't ignored by other elements.
  • Absolute
    Absolute positioning is also used to nudge an element out of its normal position. However, when an element is absolutely positioned, it leaves document flow and is ignored by other elements. Absolutely positioned elements positioned relative to the next parent container with a positioning other than static. If every parent container is statically positioned, the absolutely positioned element is positioned based on the root <html> element.
  • Fixed
    Fixed positioning positions an element relative to the root <html> element. The element is stuck on one position on the page and does not move, regardless of scrolling. Note, Fixed elements are removed from document flow.
  • Sticky
    Sticky positioning keeps an element in document flow. It sticks an element to a specified offset once a page scrolls to that offset.

Elements

  • z-index
    Specifies the appearance of an element when it overlaps with another using CSS positioning
  • top
    Specifies offset (using CSS units) from the top
  • right
    Specifies offset (using CSS units) from the right
  • bottom
    Specifies offset (using CSS units) from the bottom
  • left
    Specifies offset (using CSS units) from the left

Display Types

  • Inline
    Elements that have a box that wraps tighly around their content. They don't take a new line and are not affected by CSS width & height properties.
  • Block
    Elements that are not displayed on the same line as other elements. They take up the entire width of the page and are as tall as necessary to accomdate their content.
  • Inline-block
    Elements that are displayed inline with eachother, but behave as a block. Width and height properties affect these elements, but again, are displayed inline with other elements.

Floating

  • Float
    Elements can be floated to the side of other elements within a container. This is done using the float property, and is typically used to wrap text around an image.
  • Clear
    CSS provides a feature that prevents floating elements from being displayed inline with eachother. The specified value defines what margin needs to be clear of floats. (
