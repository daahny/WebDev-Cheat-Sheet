CSS Selectors

Selector Example Description
.class .container Selects all elements part of the container class
#id #alpha Selects all elements with an id of alpha
element element div p Selects all <p> elements inside <div>
element > element div > p Selects all <p> elements with a direct parent of <div>
element + element div p Selects the first <p> element placed after all <div> elements
element ~ element p ~ ul Selects every <ul> element that preceeds a <p> element
element, element div, p Selects <div> and <p> elements1
[attribute] [target] Selects all elements with a target attribute
[attribute] [target=_blank] Selects all elements with a target attribute equal to _blank
* * Selects all elements
::after | ::before p::before Selects just before every <p> element. Used to add content.
:not(selector) :not(p) Selects every element that's not a <p> element
