CSS Box Model

Every single element in an HTML document is modeled as a box. This can be seen by applying a border to any element.


    4 Areas of the box model

  • Content: The content is the actual content of the HTML element.
  • Padding: The padding defines the space between the content and the border.
  • Border: The border is a line that surrounds the content and padding.
  • Margin: The margin is the space encapsulating the border, padding, and content. This collapses vertically with other margins.

    Box Model Elements

  • padding: top right bottom left
  • border: style size color
  • margin: top right bottom left

Absolute Units

  • cm: centimeters
  • mm: millimeters
  • in: inches
  • px: pixels (1/96 of 1 in)
  • pt: pt (1/72 of 1 in)
  • pc: picas (12 pt)

Relative Units

  • em: Relative to the font-size of the element
  • rem: Relative to the root element font-size
  • vw: Relative to 1% of the width of the viewport*
  • vh: Relative to 1% of the height of the viewport*
*Viewport = browser window size*
*A viewport of 50cm wide, 1vw = 0.5cm*
