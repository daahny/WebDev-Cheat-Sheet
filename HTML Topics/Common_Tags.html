Types of elements

Sectioning Elements are elements that define a major area within a document

Grouping Elements are elements that organize similar content into a distinct group

Text-level Elements are element that exist within grouping elements

Sectioning Elements

<html>

  Defines the root of an HTML document and acts as a container for all other elements

<head>

  Defines meta information for a document

<header>

  Contains opening content that introduces an article or a section

<article>

  Specifies self-independent, self-contained content

<aside>

  Specifies content aside from the main content

<section>

  Marks content that shares a common theme or purpose

<address>

  Marks contact information

<nav>

  Marks a list of hyperlinks


Grouping Elements

<div>

  Marks a generic grouping of elements within the document

<figure>

  Contains an illustration, photo, or diagram. <figcaption> marks the appropriate caption.

<hr>

  Marks a break. Often empty and styled by CSS.

<pre>

  Contains a block of pre-formatted text.


Text-level Elements

<span>

  Acts as an inline-container used to mark text

<b>

  Marks text to draw attention to, usually in bold

<u>

  Marks text that's different, usually underline

<em>

  Emphasized text, usually in italics
