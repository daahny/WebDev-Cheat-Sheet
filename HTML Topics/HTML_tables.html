Table Elements
Element Description
<table> Marks a table
<thead> Marks a table header, contains headings marked by <th>
<tbody> Marks table content, where tr>s are placed
<tfoot> Marks a table footer
<th> Marks a table heading, used in <thead> and tr>
<tr> Marks a table row
<td> Marks a table data entry 
<caption>
 Marks the table title

Table Attributes
Attribute Description
scope Defines what a 
<th>
will encompass; 
row
means the 
<th>
will be a row header
rowspan Defines how many rows a 
<td>
will span
colspan Defines how many columns a 
<tds>
will span

Example

<table id="table_example">
<thead>
    <tr>
        <th scope="col">Company Name</th>;
        <th scope="col">Number of Items to Ship</th>;
        <th scope="col">Next Action</th>;
    </tr>
</thead>
<tbody>
    <tr>
        <td>Adam's Greenworks</td>
        <td>14</td>
        <td>Package Items</td>
    </tr>
    <tr>
        <td>Davie's Burgers</td>
        <td>2</td>
        <td>Send Invoice</td>
    </tr>
    <tr>
        <td>Baker's Bike Shop</td>
        <td colspan="2">3</td>
        <td>Send Invoice</td>
    </tr>
    <tr>
        <td>Miss Sally's Southern</td>
        <td rowspan="2">4</td>
        <td>Ship</td>
    </tr>
    <tr>
        <td>Summit Resort Rentals</td>
        <td>4</td>
        <td>Ship</td>
    </tr>
    <tr>
        <td>Strike Fitness</td>
        <td>1</td>
        <td>Enter Order</td>
    </tr>
</tbody>
<tfoot>
    <td>Total</td>
    <td>28</td>
</tfoot>
</table>
